# 공공 채용공고 조회 서비스

## 프로젝트 개요
공공데이터 API를 활용하여 채용공고를 조회하고,
IT 직무 + 신입 지원 가능 공고만 필터링하는 서비스입니다.

## 사용 API
- 공공데이터포털 공공기관 채용정보 API

## IT 직무 판단 기준
다음 키워드가 직무 분야(ncsCdNmLst)에 포함되면 IT 직무로 판단합니다.
- 정보
- 전산
- IT
- 소프트웨어
- 시스템
- 데이터
- 보안
- 정보화
- 개발자
- DB
- 서버
- 네트워크
- SW
- 프로그래밍
- 정보통신
- ICT

## 신입 지원 가능 기준
- "신입"
- "신입·경력"
  (경력만 모집하는 공고는 제외)

## 현재 구현된 기능
공공데이터포털 채용정보 OpenAPI 연동
IT 직무 키워드 기반 필터링
신입 지원 가능 공고 필터링 (신입, 신입·경력)
