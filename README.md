# DNY (Dream Next You) - IT 신입 공공 채용 큐레이션 서비스

공공데이터포털 API를 활용하여 공공기관의 IT 직무 신입 채용 정보만을 선별하여 제공하는 웹 서비스입니다.

## 주요 기능

### 1. 고성능 데이터 수집 및 트래픽 최적화
- 병렬 수집 (Parallel Processing): Java Parallel Stream을 활용하여 대량의 공고 데이터를 단시간에 수집합니다.
- 트래픽 절약 로직: 수집 시작 시 첫 페이지만 우선 확인하여, 신규 공고가 있을 때만 전체 수집을 진행하는 지능형 업데이트 방식을 적용했습니다.
- 비동기 처리 (@Async): 서버 시작 시 데이터 수집 작업이 서비스 구동 속도에 영향을 주지 않도록 백그라운드에서 실행됩니다.

### 2. 스마트 검색 및 데이터 필터링
- 통합 검색 시스템: JpaSpecification을 도입하여 검색어, 지역, 채용구분 등 다중 필터를 하나의 API 엔드포인트에서 처리합니다.
- IT 신입 필터링: 공고 제목 키워드 및 채용 유형 분석을 통해 IT 직무와 신입 지원 가능 여부를 자동으로 판별합니다.
- 공고 자동 관리: 오늘 기준 마감된 공고를 검색 결과에서 제외하고, DB에서 정기적으로 삭제하여 데이터 선도를 유지합니다.

### 3. 사용자 환경 및 마이페이지
- Vue.js 기반 UI: Vanilla JS 구조를 Vue.js(CDN 방식)로 리팩토링하여 데이터 바인딩과 화면 전환의 효율성을 높였습니다.
- 스크랩 기능: 관심 있는 공고를 사용자의 마이페이지에 저장하고 관리할 수 있는 북마크 시스템을 제공합니다.
- 페이징 및 정렬: Spring Data Pagination을 적용하여 대량의 공고를 효율적으로 조회하며 최신순 및 마감 임박순 정렬을 지원합니다.

## 기술 스택
- Backend: Java 17, Spring Boot 3.4.x
- Frontend: Vue.js 3 (CDN), HTML5/CSS3
- Database: JPA (Spring Data JPA)
- API: 공공데이터포털(잡알리오) 채용정보 오픈 API

## 프로젝트 구조
- controller/: REST API 요청 처리 및 응답 반환
- service/: 비즈니스 로직 처리 (병렬 수집, 필터링, 트래픽 최적화)
- repository/: 데이터베이스 접근 및 동적 쿼리 처리
- dto/: 데이터 전송 객체 (API 응답 규격 정의)
- entity/: 데이터베이스 테이블 매핑 클래스
